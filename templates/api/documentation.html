Добро пожаловать в документацию нашего API

Вы можете воспользоваться готовым REST API для работы с объявлениями. Ниже приведены доступные методы:

  • GET "api/announcement/" - вернет все записи и информацию о них.
  • POST "api/announcement/" - создает новую запись. Необходимо предоставить в теле запроса 'user_id', 'title', 'description'. Для выполнения этого действия требуется аутентификация как администратор (введите логин и пароль администратора).
  • POST "api/announcement/<announcement_id>/like" - поставит лайк записи или заберёт лайк, если применить повторно.
  • GET "api/announcement/<announcement_id>/" - возвращает информацию о конкретной записи.
  • PUT "api/announcement/<announcement_id>/" - обновляет запись с данным ID. Необходимо отправить все данные записи.
  • PATCH "api/announcement/<announcement_id>/" - частично обновляет запись с данным ID. Можно отправить только изменяемые поля.
  • DELETE "api/announcement/<announcement_id>/" - удаляет запись с данным ID.
